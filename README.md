# LightWeightLogger——一个用C语言开发的轻量的日志框架
## 使用方法
### 输出调试信息  
```C
Debug("This is a debug message.");
```
### 输出一般信息  
```C
Info("This is a info message.");
```
### 输出警告信息  
```C
Warn("This is a warning message.");
```
### 输出错误信息  
```C
Error("This is a error message.");
```
### 输出严重错误信息  
```C
Fatal("This is a fatal message.");
```
